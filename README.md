
This simple program offers all Pali texts from GRETIL to be interlinked with
DPD data for relatively easy and up to date comprehension.

GRETIL data is located here:

https://gretil.sub.uni-goettingen.de/gretil.html

Digital Pali Dictionary is located here:

https://www.dpdict.net/

All credit goes to the actual GRETIL and DPD authors. Please donate to and support them, not me.
